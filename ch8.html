Briefly desribe or define:
  • Information Security - Protecting data from those who do not have permission to access it
    		•
  • Authentication - The process of verifying that you really are the person who has the right to access a particular device
    		•
  • Authorization - Governs what an authenticated user is allowed to do
    		•
  • Dual authentication - The user enters his or her user ID and password, then uses a image with disorting letters and the user has to type the corect text into the textbox.
    		•
  • Social engineering - Process of using people to get the information you want
    		•
  • Virus - computer program that, like a biological virus, infects a host computer and then spreads
    		•
  • Worm - It can send copies of itself to other nodes on a computer network without having to be carried by an infected host file
    		•
  • Trojan horse - A computer program that does some legitimate computational tast but also, unbeknownst to the user, contains code to perform the same kinds of malicious attacks ad viruses and worms-corrupt or delete files, slow down the computer, and the like
    		•
  • Ransomware - Invades a user's computer, often by means of infected email attachment. It then encrypts the files on that computer, usually with strong enough encryption algorithm that only the author of the malware software, who holds the decryption key, can unlock the files. It then demands for money as pay to unlock the user's computer again
    		•
  • Phishing - A practice used to illegally obtain sensitive information such as credit card numbers, account numbers, and passwords
    		•
  • White hat - Security experts who try to keep us safe in cyberspae
    		•
  • Black hat - Bad guys who try to ferret out computer system weaknesses for financial gain, control, or outright destruction
    		•
  • Encryption/decryption - Make information meaningless even if someone does manage to steal it
    		•
  • Cryptography - A message is encoded before it is sent, for the purpose of keeping its content secret if it is intercepted by the wrong parties
    		•
  • Symmetric encryption algorithm - Requires the use of a secret key known to both the sender and the receiver
    		•
  • Caesar cipher - Involves shifting each character in the message to another character some fixed distance farther along in the alphabet
    		•
  • Block cipher - A group or block of plaintext letters get encoded into a block of ciphertext, but not by subsitiuting one character at a time for each letter
    		•
  • Data Encryption Standard (DES) - A encryption algorithm developed by IBM in the 1970s for the U.S. National Bureau of Standards, and is certified as an international standard by the ISO
    		•
  • Public key system - The encryption key messagees to go to a particular receiver is broadcast to everyone, but the decryption key cannot be derived from it and is known only by the receiver
    		•
  • Embedded computers - Computational devices such as chips, processors, or computers that are embedded within another system
    		•
