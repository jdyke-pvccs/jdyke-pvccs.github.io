  • Systems software-A collection of computer programs that manage the resources of a compputer and faciliates access to those resources

  • User interface-Gives the user an intuitive visual overview as well as graphical control
    		•
  • Language service-Programs called assemblers, compilers, and interpreters, allows you to write programs in a high-level, user-oriented language rather than machine
    		•
  • Memory managers-Programs allocate memory space for programs and data and retrieve this mrmory space when it is no longer needed.
    		•
  • Information managers-Program that can handle the organization, storage, and retrieval of information on storage devices

  • I/O-Software packages allow you to easily and efficiently use the many different types of input and output devices that exist on a modern computer system
